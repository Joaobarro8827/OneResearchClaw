## 🚀 How to start a research task

The researcher relies on clear instructions to function. Open the application from your desktop shortcut and follow this flow.

### Define your goal
Type your research question into the main text box. Be specific. Instead of typing "find info on batteries," type "compare the energy density of solid state batteries versus lithium ion batteries in 2024." Specific prompts lead to better results.

### Assign limits
The system allows you to set parameters. You can choose how many sources the agent scans. Keep the default settings if you want a broad overview. Change these settings if you need deep analysis from a limited set of high-quality journals.

### Launch the agent
Click the "Start Research" button. The software triggers an autonomous agent. You will see a status bar that shows the progress. The agent performs web searches, reads documents, and organizes findings. Do not close the window while the agent moves through the research workflow.

### Review the output
The application saves a report once the agent completes the work. The text appears in the center window. You can export this report as a PDF by clicking the "Save Report" button at the bottom of the screen.

## 💡 Troubleshooting common issues

If the application stops during a task, consider these adjustments.

### Connectivity problems
The agent requires a stable internet signal. If the progress bar stays at zero, check your browser to confirm the connection works. Disable VPN software if the app fails to connect to the research databases.

### Performance delays
The software handles large volumes of data. If your system runs slowly, close other programs like browsers or video editors. These programs consume the memory the agent needs to synthesize information.

### Update the software
New versions resolve performance hiccups. Visit the download page regularly to check for updates. If you experience unexpected closes, perform a fresh install by removing the old version first.

## 🧠 Understanding the research process

OneResearchClaw uses agentic workflows to improve your daily output. Here is how the system organizes complex information.

1.  **Intent Recognition**: The model analyzes your prompt to break it into sub-tasks.
2.  **Source Identification**: The agent identifies databases and articles relevant to your topic.
3.  **Data Extraction**: The system pulls text from various materials regardless of the file format.
4.  **Synthesis**: The software writes a summary that aligns with your specific request.
5.  **Refinement**: The agent self-corrects based on internal logic to ensure the final report uses objective tones.

This process removes the need for manual copy and paste. You receive a structured summary instead of long lists of links.

## 🔒 Data and privacy

The software processes your requests locally on your machine. Your personal files remain on your hard drive. The agent only sends the specific text of your prompt to the engine provider to generate the research results. You can clear your history at any time through the "Settings" menu. This ensures that no trace of your research remains on the application after you finish your work.

## ✨ Advanced tips for users

Use these techniques to get the most from your researcher.

*   **Iterative prompting**: If the initial report lacks depth, add more context to your prompt and run the task again.
*   **Format requests**: You can ask the researcher to organize findings in a table or as a bulleted list.
*   **Source citations**: The agent references its sources by default. Check the footer of the final report to find these links for further reading.

This tool functions best when you treat the agent as an assistant. Provide clear tasks, verify the output, and use the saved reports as a baseline for your final document.
